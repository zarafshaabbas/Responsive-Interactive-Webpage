# 🌐 Responsive Interactive Webpage

## 📌 Project Overview

This project is a responsive and interactive webpage built using **HTML and external CSS**. It demonstrates proper separation of structure and styling, along with responsive design techniques to ensure the webpage looks good on all devices.

---

## ✨ Features

* Clean and structured HTML layout
* External CSS for styling
* Responsive design using media queries
* Styled image with borders and rounded corners
* Button-like link that redirects to an external profile
* Mobile-friendly interface

---

## 🛠️ Technologies Used

* **HTML5** – for webpage structure
* **CSS3** – for styling
* **Media Queries** – for responsive design

---

## 📂 Project Structure

```
project-folder/
│
├── index.html     # Main HTML file
└── styles.css     # External CSS file
```

---

## 🚀 How to Run the Project

1. Create a folder on your system
2. Save the HTML code as `index.html`
3. Save the CSS code as `styles.css`
4. Make sure both files are in the same folder
5. Open `index.html` in any web browser

---

## 📱 Responsive Design

This webpage adjusts automatically based on screen size:

* **Desktop** → Full layout
* **Tablet (≤768px)** → Adjusted font and image size
* **Mobile (≤480px)** → Smaller text and buttons for better usability
